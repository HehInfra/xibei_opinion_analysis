# 4090 单标签语义分类微调说明

本文档用于在远程 4090 服务器上训练四个单标签 MacBERT 分类器：

- `topic`：主题分类
- `emotion`：情绪分类
- `stance_target`：立场对象分类
- `stance`：立场方向分类

当前训练数据来自 DeepSeek 伪标签，不是人工金标准。训练结果评估的是模型拟合 DeepSeek 标注规则的能力。

## 1. 克隆项目

```bash
git clone https://github.com/HehInfra/xibei_opinion_analysis.git
cd xibei_opinion_analysis
```

确认训练数据存在：

```bash
ls 情感分析/data/training
```

至少应看到：

```text
topic.csv
emotion.csv
stance_target.csv
stance.csv
semantic_train_summary.md
```

## 2. 创建训练环境

推荐使用 conda：

```bash
conda env create -f 情感分析/scripts/finetune/environment.yml
conda activate xibei-semantic-ft
```

如果服务器已有合适环境，也可以只安装依赖：

```bash
pip install -r 情感分析/scripts/finetune/requirements.txt
```

4090 上要重点确认 PyTorch 能识别 CUDA：

```bash
python -c "import torch; print(torch.__version__); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'no cuda')"
```

如果 `torch.cuda.is_available()` 是 `False`，不要继续训练。需要按服务器 NVIDIA 驱动和 CUDA 版本重新安装 CUDA 版 PyTorch。

## 3. 先做烟测

先用少量样本确认代码、数据、模型下载、CUDA 都正常：

```bash
python 情感分析/scripts/finetune/train_macbert_topic.py \
  --device cuda \
  --epochs 1 \
  --batch-size 8 \
  --limit-per-split 16 \
  --output-dir 情感分析/models/finetuned/topic_smoke
```

烟测成功的标志：

- 日志里显示 `device: cuda`
- 能看到每个 epoch 的 `train_loss`、`valid_acc`、`valid_macro_f1`
- 输出目录中出现 `model.safetensors`、`config.json`、`label_mapping.json`、`training_summary.json`

烟测指标不代表模型效果，因为样本太少。

```bash

```

## 6. 输出文件

每个任务训练完成后，默认输出到：

```text
情感分析/models/finetuned/<task>/
```

每个目录应包含：

```text
config.json
model.safetensors
tokenizer.json
tokenizer_config.json
label_mapping.json
training_summary.json
```

重点看 `training_summary.json`：

- `device` 是否为 `cuda`
- `train_examples`、`valid_examples`、`test_examples` 是否符合预期
- `best_valid_macro_f1`
- `test.accuracy`
- `test.macro_f1`
- `test.per_label` 中是否有某些小类完全无法识别

## 7. 参数建议

4090 初始建议：

```text
epochs：3
batch-size：16
max-length：128
learning-rate：2e-5
num-workers：2
```

如果显存充足，可以尝试：

```text
batch-size：32
max-length：128 或 256
```

如果训练不稳定或验证集指标下降：

```text
learning-rate：1e-5
epochs：2 或 3
```

当前数据只有 1349 条伪标签样本，训练轮数不要盲目拉太高，容易过拟合。

## 8. 重要注意事项

1. 首次运行会从 Hugging Face 下载 `hfl/chinese-macbert-base`。下载后会缓存在服务器用户目录下，后续同机运行通常不需要重复下载。

2. 本项目已有的 `情感分析/models/base/Chinese_macbert_base/` 是 TensorFlow checkpoint 格式，不是当前训练脚本默认使用的 Hugging Face PyTorch 模型。正式训练建议继续用默认的 `hfl/chinese-macbert-base`。

3. 训练输出模型不提交 GitHub。`.gitignore` 已忽略 `情感分析/models/finetuned/`。

4. `topic` 和 `emotion` 相对更适合先看效果；`stance_target` 和 `stance` 受语义边界影响更明显，评估时要看每类指标。

5. 类别分布不均衡，不能只看 accuracy。优先看 `macro_f1` 和每类 precision/recall/F1。

6. 训练集、验证集、测试集都来自 DeepSeek 伪标签，所以分数不能写成“人工标注准确率”。

7. 如果服务器不能访问 Hugging Face，可以先在有网络的机器下载模型缓存，或把 `hfl/chinese-macbert-base` 下载成一个本地 Hugging Face 模型目录，再用 `--model-name /path/to/model` 指向本地目录。

8. 如果远程服务器会断开 SSH，建议用 `tmux` 或 `screen` 跑训练，并把日志重定向保存：

```bash
python 情感分析/scripts/finetune/train_macbert_topic.py --device cuda --epochs 3 --batch-size 16 2>&1 | tee topic_train.log
```

## 9. 当前最推荐执行顺序

```bash

python 情感分析/scripts/finetune/train_macbert_topic.py --device cuda --epochs 3 --batch-size 16
python 情感分析/scripts/finetune/train_macbert_emotion.py --device cuda --epochs 3 --batch-size 16

```

## 10. 训练完成后预测 40803 条全量评论

四个模型都训练完成后，运行：

```bash
python 情感分析/scripts/finetune/predict_semantic_full.py \
  --device cuda \
  --batch-size 64 \
  --output 情感分析/outputs/predictions/full_semantic_predictions.csv
```

默认会读取：

```text
情感分析/data/raw/semantic_comments_full.csv
```

默认会加载：

```text
情感分析/models/finetuned/topic/
情感分析/models/finetuned/emotion/
情感分析/models/finetuned/stance_target/
情感分析/models/finetuned/stance/
```

输出文件会保留原始评论元数据，并新增：

```text
pred_topic_label
pred_topic_confidence
pred_emotion_label
pred_emotion_confidence
pred_stance_target_label
pred_stance_target_confidence
pred_stance_label
pred_stance_confidence
```

如果模型目录不在默认位置，可以显式指定：

```bash
python 情感分析/scripts/finetune/predict_semantic_full.py \
  --device cuda \
  --topic-model-dir /path/to/topic \
  --emotion-model-dir /path/to/emotion \
  --stance-target-model-dir /path/to/stance_target \
  --stance-model-dir /path/to/stance \
  --output 情感分析/outputs/predictions/full_semantic_predictions.csv
```

先做预测烟测：

```bash
python 情感分析/scripts/finetune/predict_semantic_full.py \
  --device cuda \
  --limit 100 \
  --output 情感分析/outputs/predictions/full_semantic_predictions_smoke.csv
```

确认 smoke 文件正常后，再跑完整 40803 条。
