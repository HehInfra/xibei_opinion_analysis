# 04 复现指南与文件索引

## 1. 环境说明

项目主要使用 Python、SQLite、pandas、PyTorch、Transformers、matplotlib 和 seaborn。

微调环境使用 Miniconda：

```bash
conda env create -f 情感分析/scripts/finetune/environment.yml
conda activate xibei-semantic-ft
```

如果环境已存在，可以安装或补齐依赖：

```bash
conda run -n xibei-semantic-ft python -m pip install -r 情感分析/scripts/finetune/requirements.txt
conda run -n xibei-semantic-ft python -m pip install pandas matplotlib seaborn
```

本地 MacBook Air M4 使用 MPS 跑通流程；迁移到 4090 时建议使用 `--device cuda`。

```bash
cp 情感分析/config/deepseek.example.env 情感分析/config/deepseek.env
```

```text

DEEPSEEK_MODEL=deepseek-v4-flash
```

运行：

```bash
python3 情感分析/scripts/data_prep/deepseek_semantic_annotate.py --batch-size 10 --resume
```


## 4. 单标签分类器微调

训练脚本位于：

```text
情感分析/scripts/finetune/
```

四个任务分别运行：

```bash
conda run -n xibei-semantic-ft python 情感分析/scripts/finetune/train_macbert_topic.py --device auto
conda run -n xibei-semantic-ft python 情感分析/scripts/finetune/train_macbert_emotion.py --device auto
conda run -n xibei-semantic-ft python 情感分析/scripts/finetune/train_macbert_stance_target.py --device auto
conda run -n xibei-semantic-ft python 情感分析/scripts/finetune/train_macbert_stance.py --device auto
```

在 4090 服务器上建议改为：

```bash
conda run -n xibei-semantic-ft python 情感分析/scripts/finetune/train_macbert_topic.py --device cuda --batch-size 16
```

模型输出：

```text
情感分析/models/finetuned/
  topic/
  emotion/
  stance_target/
  stance/
```

每个模型目录中包含：

```text
config.json
model.safetensors
tokenizer.json
tokenizer_config.json
label_mapping.json
training_summary.json
```

## 8. 注意事项

1. 远程服务器如果无法访问 HuggingFace，应提前把 `hfl/chinese-macbert-base` 下载到本地，并使用本地模型路径。
2. 4090 上训练时可以提高 batch size，但建议先用 `--limit` 或少量 epoch 烟测。
3. 当前模型基于伪标签训练，结论适合趋势分析，不适合宣称人工金标准准确率。
4. 小类标签样本较少，解释时应聚焦大类。
5. 如果要写论文，建议补一个人工复核集，至少覆盖每个主要标签 20-50 条。
